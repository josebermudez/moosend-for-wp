== Description ==

## Getting started    

Install the plugin. Now start growing your list!

## Moosend Subscription Forms Features


* Connect with your Moosend account in seconds
* Live preview your forms before publishing them
* Style your sign up forms using our style editor
* Create, Edit and Manage your subscription forms
* You can have multiple forms connected to different mailing lists
* Choose the custom fields you want in your form
* Create beautiful popup forms and choose when you want them showing up

###Connect To Your Moosend Acount

Simply type in your API Key and you can start generating leads.

You don't have a Moosend account yet? Get started right away by signing up for free on moosend.com.

###Create your own Forms

Just fill up the field with your API Key and you are ready to go.

You don't have a Moosend account? You can create one in just a few seconds by using our sign up form.


== Installation ==

1. Upload the entire `moosend-for-wp` folder to the `/wp-content/plugins/` directory.

2. Activate the plugin through the 'Plugins' menu in WordPress
